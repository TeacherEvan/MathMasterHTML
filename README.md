## ğŸš€ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installations required

### Quick Start

1. **Clone the repository**:
```bash
git clone https://github.com/TeacherEvan/MathMasterHTML.git
```

2. **Navigate to the application**:
```bash
cd MathMasterHTML/MathMaster-Algebra
```

3. **Launch the application**:
   - Open `index.html` in your web browser, or
   - Use a local server (recommended):
```bash
python3 -m http.server 8000
# Then visit http://localhost:8000
```

### How to Use

1. **Welcome Screen**: Click anywhere to begin your mathematical journey
2. **Level Selection**: Choose your difficulty level based on your current skills
3. **Problem Solving**: Work through problems with step-by-step guidance
4. **Progress**: Track your advancement through each level
